How to apply CSS to HTML

To apply CSS to HTML externall you have to link it in the head of your document. The code you have to add is <link rel="stylesheet" href="style.css">. This will allow you to add style to your html page.

How to Use JavaScript in a Webpage

To link JavaScript exteranlly you have to add this link to the head or body of your page.
The code is <script src="myScript.js"></script>
To add JavaScript into your html code you have to use the script tag either in the head or body tag.
An example of this is <head> <script> function myFunction() { document.getElementById("demo").innerHTML = "Paragraph changed."; } </script> </head>.
